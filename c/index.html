程序基本结构​

# include <iostream>
using namespace std;
int main(){
    return 0;
}

注意​

  • c++代码中的语句要以分号结尾，不然会报错

  • 代码中的字符要用英文字符，中文字符会报错

  • #include <XXX>

    • 表示引入某个头文件
    • 通过引入头文件，我们就能使用头文件中的某些功能，例如“iostream”使我们能够使用“cin、cout”输入输出

  • using namespace XXX;

    • 表示声明命名空间
    • 代码中使用标准库中的某些功能不需要指定完整的命名空间路径，例如声明“std”之后无需std::cout<<XXXX，而是直接cout<<XXXX
  •   int main(){
          return 0;
      }
    • 是C++程序的入口
    • 程序开始执行的地方
    • 在一个C++程序中，必须有且只能有一个 main 函数
    • return 0;返回整数0，终止程序

注释​

  • //后面的句子会被c++编译器忽略，称为注释
  • 注释的作用是对代码进行额外的备注说明，方便其他人看懂你的代码

cout​

# include<iostream>
using namespace std;
int main(){
  cout<<"hello";//输出 hello，不换行
  cout<<"c++"<<endl;//输出c++，换行
  cout<<"你好"<<"c++";//连续输出你好c++
  return 0;
}
  • cout<<输出语句，后面跟着输出的内容
  • 需要<iostream>的头文件，才能使用
  • endl表示结束输出，有换行效果
    • \n换行
    • \t一个Tab键的空格
  • cout<<XX<<XX<<XX<<....可以实现连续输出

变量​

  • 程序运行期间值可以改变的量叫做变量
  • 变量包含三个部分：变量类型、变量名和值

变量类型​

  • int 整数类型，例如1，2，12，-1
  • float 浮点数类型，也就是小数，例如0.1，0.5
  • double 双浮点数类型，也就是小数，精度比float高
  • char 字符类型，也就是单个字符，需要套个单引号，例如'a'，'@'，'1'
  • string 字符串类型，多个字符组成，需要套个双引号，例如"12223"，"@123%321afv"
  • bool 布尔值类型，只有两个值，true跟false

变量名​

  • 只能由字母、数字和下划线组成
  • 第一个字符不能为数字
  • 不能为中文

值​

  • 可以自由选取变量的值，但值要符合变量类型，不能超出变量类型所属的范围，否则会报错

使用变量​

  • 声明变量
    • 在使用变量之前要对变量进行定义，也就是要声明变量
    • 格式：
      • 变量类型 变量名;
      • 变量类型 变量名=值;
  • 使用变量
    • 直接在代码中变量名就能拿到变量对应的值

表达式​

  • 像这种有运算符参与的句子我们就叫它表达式

  • 例如：

    800*12
    12+13
    a*b
    c=e
    c=e/12

赋值语句​

  • 格式：XXX=XXX
  • =也是运算符，所以赋值语句也叫赋值表达式
  • 它的作用是把右边的值赋给左边的值
  • a=200+20+30就是把250赋给a，此时a就等于250

示范代码​

# include <iostream>
using namespace std;
int main(){
    int a=10;//声明变量
    a=a+10;//赋值表达式，把20赋值给a
    cout<<a;//20
    return 0;
}

cin语句​

# include<iostream>
using namespace std;
int main(){
  int a;
  cin>>a;//输入变量a的值
  return 0;
}

运算符​

算术运算符​

  • + - * / 熟悉的加减乘除+%取余
  • a%b取余，求a/b的余数是多少？
    • 如果一个数对2取余等于0，说明该数是个偶数
    • 如果一个数对2取余不等于0，说明该数是个奇数

复合运算符​

  • 符合XXX=XXX算术运算符YYY可以用复合运算符
  • a=a+10;可以简写成a+=10;，运算符提前，省略一个变量名
  • 同理我们有-=,/=,*=,%=

关系运算符​

  • > < >= <=大于、小于、大于等于、小于等于
  • ==等于，需要用到两个等于号
    • =一个等于号表示赋值，把右值赋给左值
  • !=不等于
    • !表示'非',非等于就是不等于

逻辑运算符​

  • &&与，两个条件都成立，才为true
  • ||或，只要一个条件成立，就为true
  • !非，条件不成立，就为true

补充​

  • 输出!5为什么结果为0
  • 对数值进行逻辑运算，会将数值先转换成布尔值，规则：非零数值转换为true，零转换为false
  • !5变成!true，于是为false
  • c++输出布尔值时，会将ture输出为1，false输出为0，于是输出0
    • c++ 的布尔类型实际上是一种整数类型，在内部存储时使用 1 来表示 true，使用 0 来表示 false。

if​

基本结构​

  if(条件){//if翻译为如果，条件成立，执行if花括号的内容
    cout<<"条件成立";
  }else{//else翻译为否则，条件不成立，执行else花括号的内容
    cout<<"条件不成立";
  }
  • if后面跟一个条件，要用()包含条件
  • 花括号里的内容要缩进一个Tab键（四个空格）

条件​

  • 条件表达式可以有两种形式：关系表达式和逻辑表达式
  • 关系表达式，例如用到关系运算符的：5>10、2==2等
  • 逻辑表达式，例如用到逻辑运算符的：5>10 && 2==2 、!5等

多个if​

  • 1.可以直接重复写多个if
  if(条件){

  }
  if(条件){

  }
  • 2.else后面要跟条件，需要多加个if
    • else表示否则，是对前面条件的否定，所以要前面条件都不成立，才会去判断该条件。
    • 也就是说，if-else if的，一旦有一个条件成立，就会直接跳过后面所有条件。
  if(条件){

  }else if(条件){

  }else if(条件){

  }else{

  }

多重if​

  if(条件1){
    if(条件2){

    }else{

    }
  }else{

  }
  • 像这样一个if套着另外一个if的，我们叫它if嵌套
  • 代码先去判断条件1是否成立，如果条件1成立，它才会去判断条件2
  • 相当于 if(条件1 && 条件2)

随机数​

#include <iostream>
#include <cstdlib> //引入c++内置函数，能使用rand()，srand() 
#include <ctime> // 引入时间模块，能使用time(0) 
using namespace std;
int main(){
    srand(time(0));
    int a=rand();
    return 0;
}

srand()​

  • 种子对应英文是“seed”，s+rand就是随机种子
  • 功能：设置随机种子
  • 原理：为rand()设置初始值，在()里填写初始值
  • 注意：
    • srand(20)初始值一直是20，导致每次运行代码都是一样的结果
    • srand(无时不刻一直在改变的量)才能让我们每次运行代码结果不一样

rand()​

  • 随机对应英文是“random”
  • 功能：生成范围在“0-无穷大”的随机整数
  • 原理：依据初始值通过某些运算生成随机数
    • 由于它是通过运算来获得随机数，所以不是真正的随机数，我们也称它"伪随机数"
    • 它需要初始值，也就是随机种子
  • 生成[a,b]范围的随机数：rand%(b-a+1)+a

time(0)​

  • 功能：获得当前时间
  • srand(time(0))时间不断改变，初始值不断改变，随机数就不断改变
